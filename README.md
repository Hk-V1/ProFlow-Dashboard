# ProFlow - Personal Productivity Dashboard

A full-featured personal productivity and work management app built with Streamlit.

---

## Features

| Section | What you can do |
|---|---|
| Auth | Signup / Login with secure hashed passwords |
| Dashboard | Today's tasks, upcoming deadlines, quick stats |
| Tasks | Add/Edit/Delete with priority, category, deadline, status |
| Students | Manage students, fees, schedules, mark attendance |
| Companies | Track clients/companies and their associated tasks |
| Calendar | Add events (meetings, classes, deadlines) |
| Budget | Income/Expense tracker with pie chart & trend line |
| Daily Planner | Quick daily to-do list with progress bar |
| Settings | Edit profile, change password, export all data |

---

## Database
SQLite (`proflow.db`) is auto-created on first run. All data is stored locally.

---

## Tech Stack
- **Streamlit** - UI framework  
- **SQLite** - Local database  
- **Pandas** - Data handling  
- **Plotly** - Charts & visualizations
